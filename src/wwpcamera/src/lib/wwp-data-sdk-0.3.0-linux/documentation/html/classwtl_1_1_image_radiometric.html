Detailed Description

Derived Class representing thermogram with temperature data and thermal parameters included.

Provides interface for extracting and saving data to radiometric images. Instances are created only trough functions in class Center. A radiometric JPEG is a standard JPEG with a FFF-file inserted in the APP1-tag
